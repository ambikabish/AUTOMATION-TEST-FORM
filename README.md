Automation Test Form
Overview

Automates filling out web forms using Python & Selenium. Saves time, reduces manual errors, and demonstrates practical web automation skills.

Features

Fill text fields automatically

Select dropdowns, checkboxes, and radio buttons

Submit forms automatically

Easy to customize for different forms

Tech Stack

Python

Selenium WebDriver

Usage

Clone the repo:

git clone https://github.com/yourusername/automation-test-form.git


Install dependencies:

pip install -r requirements.txt


Update automation_form.py with your form data and URL.

Run the script:

python automation_form.py
